## Snake Monster

This Snake Game was developed as a fun side project after I came across similar versions online and got curious about how they work. 
The simplicity and nostalgia of the game motivated me to recreate it from scratch using JavaScript.
While building this, I focused on implementing core game mechanics such as smooth movement, real-time controls, collision detection, and score tracking.
It was a great way to strengthen my understanding of JavaScript fundamentals, DOM manipulation, and event handling.
Overall, this project reflects my curiosity-driven learning approach and my interest in building interactive and engaging web applications.
